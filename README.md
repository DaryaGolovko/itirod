# Golovko Darya, 053502

Main functions:

1. Authorization
  - Sign in option for users.
  
2. Registartion
  - Sign up option for new users.
  
3. Messaging
  - Send and receive messages
  
4. QRcode 
  - The ability to search people with the help of qr code.

Models

1) User model
  - ID
  - Username
  - Password
  - Phone number
  
2) Chat model
  - ID
  - Name
  - List of users

3) Message model
  - ID
  - User ID
  - Message text
  - Time
  - Chat ID
